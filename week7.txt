Linked Lists 

    คือ โครงสร้างข้อมูลที่ถูกออกแบบมาใช้เก็บข้อมูล โดยที่ข้อมูลแต่ละตัวมีการเชื่อมถึงกัน เพื่อให้ผู้ใช้สามารถเข้าถึงข้อมูลแต่ละตัวได้
    Array ก็เป็นโครงสร้างข้อมูล

    Structure 

        คือ การกำหนดรูปแบบโครงสร้างข้อมูลที่มีการรวบรวมตัวแปรที่เก็บข้อมูลหลายเรื่องที่สอดคล้องกัน
    
    การประกาศตัวแปรประเภท Structure

        ประกอบด้วย Structure  ชื่อตัวแปร ;
        การเข้าถึงสมาชิกแต่ละฟีล ทำได้โดยใช้ . (dot operator)

    ตัวแปร pointer ไปยัง Structure

        นอกจะใช้การใช้ . (dot operator) แล้วสามารถใช้ --> (structure pointer operator) หรือเรียกอีกอย่างว่า arrow operator
        
    การใช้ typedef

        กำหนดประเภทข้อมูลขึ้นมาเพิ่มเติม เพื่อให้กลายเป็นข้อมูลใหม่ที่ใช้เพียง 1 คำ แทนประเภทข้อมูลได้

    Self-Referential Structures

        เป็นโครงสร้างที่ออกแบบมาจัดการ linked lists โดยสร้างจาก structure ภายในมี pointer ที่อ้างอิงกลับไป structure

        ใช้คำว่า node ในการอ้างถึงหน่วยที่เก็บข้อมูล 1 หน่วย

    การเข้าถึงข้อมูล linked list (Traversal)

        การสร้างตัวแปรเพิ่ม เพื่อใช้ในการเข้าถึง list การเข้าถึง list จะเริ่มขึ้นที่ต้นของ list ที่อ้างถึงไว้ตำแหน่งแรก 
        แล้วเคลื่อนไปตามการเชื่อมโยงตำแหน่งต่างๆที่กำหนด แล้วนำข้อมูลขึ้นแสดงไปเรื่อยๆจนจบ list

    
    การลบข้อมูลออกจาก linked list (Deletion)

        จะต้องมีกำหนด node ที่ต้องการจะลบ และเก็บตำแหน่งโหนดก่อนหน้าที่ต้องการจะลบ โดยการเข้าถึงข้อมูลใน list 

        ลบตำแหน่งต้น list ให้เลื่อนตำแหน่งต้นไปยังโหนดถัดไป และเปลี่ยนลิงค์ก่อนหน้าให้เป็น Null

    การเพิ่มข้อมูลเข้า linked list (Insertion)

        การให้ head ชี้ไปที่ null เมื่อใน list ไม่มีข้อมมูลเลย
        ถ้ามีข้อมูลใน list ให้เชื่อม node ใหม่เข้า list  และปรับให้ head หรือ tail ชี้ไปที่ node ใหม่ที่เอาเข้ามา
