Array 

    คือ กลุ่มตัวแปรที่มีประเภทเดียวและถูกจัดมาร่วมกัน

    การประกาศ Array

        ประกอบด้วย ประเภทตัวแปร ชื่อตัวแปร และขนาด(Size)ที่ถูกครอบด้วย []
        เช่น int Array[10];

    การเรียกใช้ Array

        ถ้าต้องการเรียกใช้ตัวแปรใน Array ซึ่งจะมีเบอร์กำกับ หรือ (index) ซึ่ง index จะเริ่มที่ 0 
        เช่น Array[10] จะสามารถเรียกใช้ได้ตั้งแต่ Array[0]-Array[9]

    การตั้งค่าเริ่มต้นของ Array

        เช่น Array[5] = {0, 1, 2, 3, 4} 
        แต่ถ้ามีการประกาศ Array[10] = {0, 1, 2, 3, 4} แต่ Array มีเพียง 5 ตัว ส่วนที่เหลือจะเป็น 0  

        สามารกำหนดขนาด Array จากจำนวนค่าเริ่มต้นได้ เช่น Array[] = {0, 1, 2, 3, 4} Array จะมีขนาดเท่ากับ 5

    Array สองมิติ

        เป็นตัวแปรที่มี Row(แถว) กับ Column(หลัก) มีรูปแบบเป็นตาราง ที่มีการแสดงตำแหน่ง 2ตัว
        จะต้องมีการบอกขนาด column เช่น Array[][3]
        สามารถใช้ตัวแปรบอกขนาด Array ได้ 
    
    การส่งตำแหน่ง

        Array เป็น Address ซึ่งการจะเรียกตัวแปรที่เก็บตำแหน่งว่า pointer

    ส่ง Array สองมิติไปให้ฟังก์ชัน

        ใน parameter จะต้องมีการบอกขนาดของ column เช่น int array(int Array[][3], int row, int column)
